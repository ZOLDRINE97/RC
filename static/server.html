Web Server (Sockets)

O que este servidor faz

  • Escuta ligacoes TCP em 127.0.0.1:8080.
  • Recebe pedidos HTTP e processa apenas o metodo GET.
  • Serve ficheiros da pasta static.
  • Se o ficheiro existir, responde com 200 OK.
  • Se nao existir, responde com 404 Not Found.
  • Se o metodo nao for GET, responde com 501 Not Implemented.

Seguranca basica

O servidor limita o acesso a ficheiros dentro da pasta static. Assim, evita devolver ficheiros fora da pasta publica.

Como testar

  • Abrir http://127.0.0.1:8080/server.html
  • Abrir http://127.0.0.1:8080/index.html
  • Tentar um ficheiro que nao exista para receber 404
